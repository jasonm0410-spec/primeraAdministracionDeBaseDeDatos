# PrimeraAdministracionDeBaseDeDatos


 #👋¡Hola! Bienvenido a mi proyecto.

Este proyecto consiste en el desarrollo de una base de datos para una zapatería, realizada como parte de una práctica utilizando MySQL y phpMyAdmin. La base de datos cuenta con 5 tablas principales: Clientes, Empleados, Productos, Ventas y Detalle de Ventas, las cuales están relacionadas mediante claves primarias y foráneas para mantener la información organizada.

El objetivo de este proyecto es aplicar los conocimientos sobre la creación de tablas, inserción de datos y relaciones entre tablas en una base de datos relacional.
